使用油猴脚本进行自我约束

使用油猴脚本来对我的行为进行约束，避免无意识地刷视频、刷图片浪费光阴。

油猴脚本简述

油猴脚本是一种 js 脚本，它能够在网页运行时执行，以修改网页内容、拦截 HTTP 请求，甚至主动发送跨域 HTTP 请求，访问用户文件系统（油猴支持file:协议）等，使得网页能够更易于使用，但代价则是可能的安全风险。油猴脚本会自己提供一些 API 供使用，油猴脚本也可以引用第三方库。

油猴脚本默认执行在一个沙箱环境中，相当于是一个虚拟的 window 对象，这个沙箱环境中部分 API 和网页的 window 对象是共享的，如 fetch，XMLHttpRequest，因此修改这些对象能够反应到网页上。

油猴脚本也能配置直接使用网页的上下文（默认配置，// @grant none），即 油猴脚本的 window === 网页的 window，使用这种方式的话因为不安全，所以不允许使用油猴自己的 API；另一种选项是使用 unsafeWindow，它就是网页的 window，把它作为一个变量供油猴脚本去访问，从而让开发者控制安全性。
