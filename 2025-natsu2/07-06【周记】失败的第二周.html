【周记】相对失败的第 2 周

本周总结

这周是失败的，从画画时间上能看到，我这周画画时间只有 5 小时（时间基本都投入到 Project One 中了），是上周的一半，虽然我付出了一些时间去进行研究，因此没有记录时间，但总时间仍旧是不行的。我还说上周是中等偏下呢，这周不更是下等？

原因我认为有：

  1. 被眼下的“胜利”冲昏了头，决心不再自学 Krenz 的课程，完成（指中道崩殂）Project One，都让我有懈怠，浪费时间。这他妈算是什么胜利？
  2. 官能小说和 B 站搞笑视频，对我影响太大，它们就是我的抖音快手，我最不屑那些短视频那些直播，但我仍旧着了它的道。
  3. 孤独，我必须承认仍旧有孤独，因为孤独我刷视频，因为孤独我浪费很多时间在聊天软件上……但在报班后大概就有能够讨论的人了，可以缓解
  4. 周末回家了，家里的氛围太容易让人放松，我都忘记了我还在计划中，想画画但都懒得把画纸掏出来
  5. 有两天都喝了酒（你究竟出于什么心态跑去喝酒的？“胜利”吗？）

我确实是没懂何为即时满足，何为延迟满足，我寻找满足感的方式，是目光短浅的，我需要研究如何调整。

要说这周有何进展的话：

  1. “圆柱大法”的视频看了个开头，感觉有一些启发，后面有兴趣给它看下去
  2. 决心不再自学 Krenz 透视（我现在开始怀疑我这个选择也有一定逃避心理），因此这阶段的安排会更加自由
  3. 研究了 PBL，后面试图以它为主要方式去安排任务
  4. Project One 进行了部分，做了一些至关重要的实践（虽然我还是中途放弃了）

Project One 那样的项目我后面就不那么干了——我要以兴趣驱动，Project One 中有背景，大透视，复杂服饰，原创人设……我现在水平又不行，同时关注这些东西让我逐渐偏离最初的设想，最终失掉了兴趣。后面的创作型项目我要从尽量简单的开始——无背景，弱透视，简单平涂服饰，同人角色，以尽量简单的题材，专注在角色、神态、动作上，优先让我画出完整的作品来——草图->线稿->上色，体验完整的二次元的工作流，而非像 Project Zero 和 Project One 那样，卡在草图上。

本阶段的项目和下周目标

下周是第 3 周了，该阶段还剩下 4 个星期。这个阶段是“结构和人体”，我是打算在这阶段定下至少这些项目（仅供参考，可以认为这里是项目池）：

  1. 各角度大头研究（笔记，设定集）
  2. 眼睛或鼻子或嘴巴结构研究（笔记）
  3. 头发一般结构研究（笔记）
  4. 透视中的立方体的研究（基于 drawabox）（笔记）
  5. 透视中的圆的研究（基于 drawabox 和 B 站某视频）（笔记）
  6. 透视立方体攻关（基于 drawabox 的挑战）（练习集）
  7. 透视圆攻关（基于 drawabox 的挑战）（练习集）
  8. 无背景黑白双人百合图创作（可重复）（作品）
  9. 表情集研究（笔记和表情集作品）
  10. 头颈连接研究（笔记）
  11. 人体动态概括研究（即动态，bean，体块等）（笔记）
  12. 人体体块和界标研究（笔记）
  13. 手的结构研究（笔记）
  14. 手臂结构、动态研究（笔记）
  15. 衣褶粗略研究（笔记）
  16. 男女老幼面甲结构研究（笔记）

下周的话，期待能够完成：

  1. 各角度大头研究
  2. 透视中的立方体的研究
  3. 透视中的圆的研究
  4. 无背景黑白双人百合图创作

以及日常任务的话期待每天做 turnaround 练习。

FIGHT！这周画了5小时，这周称的体重为65.5kg，无论哪个都是一个很好的开始——随随便便就能看到进步。
