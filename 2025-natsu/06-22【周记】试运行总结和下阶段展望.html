【周记】试运行总结和下阶段展望

本周和本阶段的总结

试运行这个星期，怎样呢？

观察画画数据，我这个星期画了七个小时，其中有 3 天接近 2 小时。周末则是基本没画（我本打算完成 Project Zero 的……），证明家里的环境确实对我有很大影响。

健身？我倒是在持续坚持下去，但同样地，周末两天我同样懈怠了。

饮食，这个需要检讨一下，五天中我有三天没有好好维护饮食，星期三星期四中午均喝酒了，星期五则是晚上喝了。这基本上影响了我一整天的状态。甚至于喝了酒后睡觉一两个小时这点都是次要的了，最主要的问题是影响了一整天的心态。

我晓得刚开始我不能对自己苛责太多，但我认为这确实是开了一个坏头，值得好好检讨和反思。我得找到我的根本问题。

根本问题在于：

  1. 我没有去以之前所提到的爱憎的感情为驱动力，这让我在情绪低落时难以自拔，以及对自己的摆烂行为采取放任态度，对它感觉舒适，而没有明确意识到它的危害
  2. 我没有主动去寻找乐趣（实践中画着画着就更容易画下去，这是被动地找到乐趣的结果），甚至于，在贯彻不容许自己找到乐趣的这样的苦修主义，这让我拒绝乐趣、拒绝放松、拒绝探索、拒绝好奇，让一切都变得灰暗
  3. 仍旧没有正确意识到煎熬的正面意义，仍旧把它当作内心中的猛兽，害怕它，而没有意识到，煎熬是成长的契机

对第一点，这就是说，我应当去主动地向自己灌输（而不是被动地等着），对躺平，对摆烂，对情绪低落，对（工作日）酗酒的恨（对的，是仇恨），同时也要灌输对奋斗，对学习，对吃苦耐劳的爱。我要主动把我的各种行为去和更深远的东西关联起来，即把自己的生活上纲上线。但同时地，这里要把握好平衡——爱和恨是相辅相成的，不可有偏颇。

但同时，在把自己的生活上纲上线的同时，又需要第二点——我不应当光盯着宏伟的目标，我同时应当在自己的学习生活中主动寻得乐趣，以促进我走下去。我当前的思想觉悟还不够，不能像主席那样“与天斗，其乐无穷，与地斗，其乐无穷，与人斗，其乐无穷”，我仍旧需要激励，而最健康的激励，就是在斗争中，在改造世界（和改造自己）的过程中寻得这种乐趣。上纲上线告诉我为什么，改造世界和自己的乐趣让我持续走下去。这里的“改造世界和自己”也可以简称为“创造”。

第三点则是字面意思，没有为跳出舒适区赋予意义（这个倒应当合到第一点，作为爱的部分，但他相对更加重要一些，值得拎出来）。

第二点或许对我来说是新东西——我在之前的计划中，有点追求苦修主义了，但这完全没有考虑到我当前的情况，我这必须折个中了。我怀疑，我的“苦修主义”有两个来源——焦虑和责任心，但怎么说呢？既然我必须关注当下才能在未来做出贡献，那我关注当下就是最负责任的行为了，但……我同时必须时刻提醒自己不要忘记初心。

还是得看实践，上面写的东西必须得落实：关于如何去给自己灌输爱和恨，这个得在实践中继续研究。

下阶段目标

首先的问题，是把上面的爱憎、乐趣，看如何落实到实践上。这个还得继续思考，但可能的方向是：

  1. 主动进行感情的动员，进行对自己的灌输
  2. 主动去在行动中寻找乐趣，时刻回想自己学习编程的经验，我是如何出于趣味去学习各种东西，做实验，去深化已经学会的东西的。在学习画画时，也要去对每一点细枝末节都保持兴趣（即使当前看不出意义），说不定背后就藏着一个新世界呢

这个得继续探索。

然后，关于下阶段的目标。因为考虑到后续要去跟随 Krenz 的透视课程进行学习，具体时间点是七月十八日结束报名，八月一日到九月十九日八周课程，我只有五个星期能自由安排。

为了保证学习的质量，我考虑，在这五周里，进行课程的自学（就当预习了），同时也要吸收其他透视资料的概念，如 drawabox，B 站上的视频。我之前看的时候好像注意到 Krenz 的课程中没有典型的结构透视的过程，他更专注完整场景？这个我得尝试补齐一下，这个主要来自于 drawabox 恐怕。

以及，我要每两个星期（或者看执行情况，如果比较容易融入到生活中的话，那就一个星期？）去进行一次完整的创作，去实践，Project Zero 本该是第一次的，但试运行期间没有做。所以，Project Zero 要顺延到这个阶段的第一周。

总之，下阶段（称为“结构和透视”），要关注这几个事情：

  1. 完成 Project Zero（在第一周开头，马上就去完成它）
  2. 吸收 Drawabox lesson 0-2 的理论甚至更多
  3. 完成 Drawabox 的作业，250 box 和 250 cylinder 挑战（如果认为有必要的话，但考虑到时间可能不充分，可能要减量）
  4. 看 Krenz 透视课程并完成所有作业，保质保量，并带着趣味去完成。
  5. 空余时间学习舞蹈（据说对绘画的空间思维也有帮助），我之前认为我应当在习惯计划之前推迟
  6. 每天记黑白名单——哪些行为是应当鼓励的，哪些行为是应当批评的，我以前总是关注后者，但忽略了前者，这得改善
  7. 编写并不断完善 《战术手册》，对各种原因、各种形式的 EMO，懈怠等小资我的进攻，要能够有可操作的应对措施

下周目标

下周是结构和透视阶段的第一周。先保守地定任务。

  1. 完成 Project Zero（预计两天完成，研究画什么也包含在内）
  2. 完成 Krenz 透视课程的第一周的学习和作业
  3. 完成 Drawabox lesson 0-1 课程和作业
  4. 倘若能超额完成（真的能有吗？？），就继续回到原来的头颈肩等地方
  5. 倘若我中途仍旧给自己整 emo 了，就去根据上面说的东西把自己重新拔起来吧，去灌输爱恨，去找到乐趣；关于找到乐趣，随便跟随视频或者图片教程学一点小把戏是一个选择
  6. 最后，每天的热身和 daily turnaround 不能停！

期待每天能画 3 小时往上，现在不像试运行那样没有明确的目标，现在有系统教程跟随了，理应能更加“不带脑子”去画了。

以及，我注意到看百合官能小说时我有比看插画更大的满足感，我在想，或许把官能小说的文字使用图像去描绘出来，能够是一个富有乐趣的练习、创作方式。
